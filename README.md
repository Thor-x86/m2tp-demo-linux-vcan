# M2TP Demo with Linux's Virtual CAN

> **WARNING:** You MUST [install M2TP](https://github.com/Thor-x86/m2tp#how-to-install) first!

## Initialize the VCAN driver

```bash
./initialize.bash
```

## Build the source code

```bash
make
```

## Run `leader` Side

```bash
./leader
```

## Run `member` Side

```bash
./member
```
